# RenewCanvas Museum Curation Open Artifact

This folder documents the rule-based AI-curator prototype used by `F08-ai-curator`.

The current version is deterministic and does not use a trained model. It arranges artworks by public metadata:

- artwork category/type
- recyclable materials
- impact score or kg diverted
- optional public tags and artist location retained for later display/search use

## Files

- `curation-schema.json`: request and successful-response schema definitions for the curation API.
- `methodology.md`: transparent placement methodology, limitations, and bias review.
